# Community Workflow Index

`index.json` 是從 [Zie619/n8n-workflows](https://github.com/Zie619/n8n-workflows) 蒸餾出的本地檢索索引,供 Agent Hub 的 AI 在建流程時引用社群同型流程當參考。

- 只含 metadata:每個 workflow 的路徑、名稱、節點型別清單、觸發型(scheduled/webhook/manual/triggered)與節點數,**不含任何原始 workflow 程式碼或憑證**。
- 來源與致謝:感謝 [Zie619/n8n-workflows](https://github.com/Zie619/n8n-workflows) 整理的社群 workflow 合集;授權以來源 repo 為準。
- 完整 workflow JSON 請循 `path` 欄位到來源 repo 取得。
